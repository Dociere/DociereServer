## Docière Server

Docière Server is the backend platform for Docière Pro, powering real-time collaboration, remote data storage, and AI integrations. It enables users to access and sync their documents from anywhere, supports concurrent editing across teams, and ensures seamless management of projects, citations, and version history. Designed to complement Docière Pro, the server provides a robust, scalable infrastructure for researchers, students, and technical writers to work efficiently in a connected environment.

### Current Version: v0.3.0 (Beta)

<hr>

### Key Features

- **Unlimited Collaborative Editing**: Now an entire team can collaborate on a project in real time.
- **AI Chat mode**: With built-in AI Chat mode you can now overcome writers block with ease.
- **Remote Database Access**: Store and access your documents from anywhere, ensuring your data is always available on the go.
